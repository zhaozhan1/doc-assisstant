### F4.1 Web 后端 API 层

**边界**：
- 基于已有模块暴露 RESTful API
- WebSocket 接口：导入进度推送、生成流式输出
- 文件上传和下载接口
- 统一错误处理和响应格式

**不做什么**：不实现新的业务逻辑，仅包装已有模块

**交付要求**：
- 所有后端功能可通过 API 访问
- WebSocket 连接可正常推送进度和流式内容
- API 文档（OpenAPI/Swagger）自动生成
- 集成测试覆盖主要 API 端点

**依赖**：F1-F3 全部功能点

---

### F4.2 前端项目初始化与布局框架

**边界**：
- 前端项目搭建（React + TypeScript + Vite + Ant Design）
- 顶部导航栏 + 页面路由
- 4 个页面的布局框架（内容区为空）
- 全局样式和组件库基础配置

**不做什么**：不做具体页面功能实现

**交付要求**：
- 可在浏览器中打开，导航栏可切换页面
- 页面布局符合设计规格
- 响应式布局适配常见屏幕尺寸

**依赖**：无（可与 F4.1 并行）

---

### F4.3 知识库页面

**边界**：
- 文件/文件夹上传（支持拖拽）
- 导入进度实时展示（WebSocket）
- 已导入文件列表表格（分页、按类型/时间筛选、排序）
- 操作：重新索引、删除、修正分类标签
- 知识库统计卡片（文件总数、类型分布、最后更新时间）

**不做什么**：不做高级搜索

**交付要求**：
- 全部功能可正常操作
- 进度实时更新，无卡顿
- 操作结果即时反映到列表

**依赖**：F4.1, F4.2

---

### F4.4 写作页面

**边界**：
- 三种模式切换：一步生成 / 分步检索 / Word 转 PPT
- 一步生成：输入框 + 生成按钮 + 流式预览 + 下载
- 分步检索：搜索框 + 结果列表（可勾选）+ 要求输入 + 生成 + 预览 + 下载
- Word 转 PPT：文件选择 + 生成按钮 + 预览 + 下载
- 生成完成后展示素材来源清单

**不做什么**：不做历史记录

**交付要求**：
- 三种模式均可在 UI 中完整操作
- 流式预览可正常显示
- 文件可正常下载

**依赖**：F4.1, F4.2

---

### F4.5 模板管理页面

**边界**：
- 模板列表展示（区分内置/自定义）
- 模板详情查看
- 自定义模板：新建、编辑、删除
- 模板编辑器：结构大纲编辑（标题层级 + 各段写作要点）

**不做什么**：不做模板预览/试用

**交付要求**：
- 内置模板可查看详情
- 自定义模板 CRUD 操作正常
- 编辑器可正常编辑结构大纲

**依赖**：F4.1, F4.2

---

### F4.6 设置页面

**边界**：
- 知识库配置：源文件夹路径、数据库路径、分块参数
- LLM 配置：Provider 选择、Base URL、模型选择、API Key
- 在线检索配置：开关、Provider、API Key、优先域名
- 输出配置：默认格式、保存路径
- 配置保存后即时生效

**不做什么**：不做配置校验/测试连接

**交付要求**：
- 所有配置项可正常编辑和保存
- 保存后配置持久化，重启不丢失
- LLM Provider 切换后对应功能可用

**依赖**：F4.1, F4.2
